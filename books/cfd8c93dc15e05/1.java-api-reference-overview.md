## そもそも Java API とは？

Java API とは、Java プログラミング言語で利用できるクラスやインターフェースの集合体のことを指します。
Java では、Java API を利用することで、様々な機能を簡単に実装することができます。

その中でも、`java.util` パッケージは、Java API の中でも特に重要なパッケージの一つです。
このパッケージには、コレクションフレームワークや日付と時刻のクラスなど、様々な便利なクラスが含まれています。

https://docs.oracle.com/javase/jp/25/docs/api/java.base/java/util/package-summary.html

今回は、`java.util` パッケージの中でも特に重要なクラスである `ArrayList` クラスを見てみましょう。

https://docs.oracle.com/javase/jp/25/docs/api/java.base/java/util/ArrayList.html

## `ArrayList` クラスとは？

`ArrayList` クラスは、可変長の配列を実装したクラスです。
このクラスを利用することで、要素の追加や削除が簡単に行えるようになります。

例えば、以下のように `ArrayList` クラスを利用して、文字列のリストを作成することができます。

```java
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();
        list.add("Hello");
        list.add("World");
        System.out.println(list); // 出力: [Hello, World]
    }
}
```

このように、`ArrayList` クラスを利用することで、簡単にリストを作成し、要素を追加することができます。

## `ArrayList` クラスの API リファレンスを見てみよう

では、`ArrayList` クラスの API リファレンスを見てみましょう。

![](/images/array-list-reference.png)

おそらく、API リファレンスを初めて見る人は、どこから見ればいいのか分からないかもしれません。

しかし、API リファレンスは、クラスの構造や利用方法を理解するための重要な情報が詰まっています。
たとえば、`ArrayList` のページには次のような情報が載っています。

- どのパッケージに属しているか
- どのクラスを継承しているか
- どのインターフェースを実装しているか
- どのようなコンストラクタがあるか
- どのようなメソッドがあるか
- 各メソッドの引数、戻り値、例外、説明

最初からすべてを理解する必要はありません。
大切なのは、「自分が使いたい機能はどこに書かれているのか」を少しずつ探せるようになることです。

たとえば、`ArrayList` に要素を追加したいなら、メソッドの一覧から `add` を探します。
要素数を知りたいなら `size` を探します。
要素を取り出したいなら `get` を探します。

```java
ArrayList<String> names = new ArrayList<>();

names.add("Alice");
names.add("Bob");

System.out.println(names.size()); // 2
System.out.println(names.get(0)); // Alice
```

このように、API リファレンスは「暗記するための資料」ではなく、「必要な情報を調べるための資料」です。

この本では、この `ArrayList` クラスの API リファレンスをもとに、オブジェクト指向プログラミングの基本概念を学んでいきます。
